# Задание 3: Архитектура push-уведомлений

## Описание
В мобильное приложение "Петрушка Зеленая" нужно добавить получение push-уведомлений:
- о забытой корзине
- об отмене заказа
- рекламные рассылки

На бэкенде — микросервисная архитектура.

## Решение
Архитектура построена с использованием **C4 model** (уровни Context и Container).

### Документы
- `c4-context-diagram.md` — диаграмма контекста (уровень 1)
- `c4-container-diagram.md` — диаграмма контейнеров (уровень 2)

## Ключевые решения
- Внешние push-сервисы: **Apple APNs** (iOS) и **Google FCM** (Android)
- Внутренний **Message Broker** (очередь) для надёжности
- Отдельный **Notification Service** для формирования и отправки пушей
- **Device Token DB** (Redis) для хранения push-токенов

# Тестовое задание: интернет-магазин "Петрушка Зеленая"

## О чем проект
В этом репозитории представлено решение тестового задания для Junior Системного аналитика.  
Задания охватывают анализ требований, проектирование API и построение архитектуры push-уведомлений.

## Структура репозитория
- `task1-requirements-analysis/` — анализ и исправление ТЗ функционала "Корзина"
- `task2-api-design/` — проектирование REST API для экрана магазинов-партнеров
- `task3-architecture/` — архитектура отправки push-уведомлений (C4 diagrams)

## Как использовать
1. Откройте любой файл `README.md` в соответствующей папке
2. Для схем C4 рекомендую открывать в браузере или draw.io
